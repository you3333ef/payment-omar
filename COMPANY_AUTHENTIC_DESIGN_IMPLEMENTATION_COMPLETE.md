### CompanyBranding Interface

Each company has a complete branding object:

```typescript
interface CompanyBranding {
  id: string;
  name: string;
  nameAr: string;
  country: string;
  logo: string;
  colors: {
    primary: string;
    primaryDark: string;
    primaryLight: string;
    secondary: string;
    accent: string;
    background: string;
    surface: string;
    text: string;
    textLight: string;
    textOnPrimary: string;
    border: string;
    success: string;
    warning: string;
    error: string;
  };
  gradients: {
    primary: string;
    secondary: string;
    accent: string;
  };
  fonts: {
    primary: string;
    primaryAr?: string;
  };
  buttonStyles: {
    primary: string;
    secondary: string;
    outline: string;
    link: string;
  };
  shadows: {
    sm: string;
    md: string;
    lg: string;
    xl: string;
  };
  borderRadius: {
    sm: string;
    md: string;
    lg: string;
    xl: string;
  };
  heroImage: string;
  ogImage: string;
  description: string;
  website?: string;
  officialColors?: string[];
}
```

## 🚀 Usage

### Using Company Branding in Components

```typescript
import { getCompanyBranding } from '@/lib/companyBranding';

const MyComponent = () => {
  const branding = getCompanyBranding('aramex');

  return (
    <div
      style={{
        background: branding.gradients.primary,
        fontFamily: branding.fonts.primaryAr,
        boxShadow: branding.shadows.lg
      }}
    >
      <button className={branding.buttonStyles.primary}>
        Pay Now
      </button>
    </div>
  );
};
```

### Getting All Companies

```typescript
import { getAllCompanies, getCompaniesByCountry } from '@/lib/companyBranding';

// Get all companies
const allCompanies = getAllCompanies();

// Get companies by country
const uaeCompanies = getCompaniesByCountry('UAE');
const saudiCompanies = getCompaniesByCountry('Saudi');
```
